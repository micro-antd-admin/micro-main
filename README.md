## 集成qiankun微前端解决方案
本项目对 antd Admin进行改造集成支持微前端的动态路由加载等微前端主应用和微应用路由交互以及状态通信。
## 使用
### clone
```bash
$ git clone https://github.com/iczer/vue-antd-admin.git
```
### yarn
```bash
$ yarn install
$ yarn serve
```
### or npm
```
$ npm install
$ npm run serve
```
更多信息参考 [使用文档](https://iczer.gitee.io/vue-antd-admin-docs)
